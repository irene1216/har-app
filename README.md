# har-app
Using Vue.js front-end framework to build a brochure style web app for Har.
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

### Remove package
Change package.json
```
npm prune <pkg> --  production
```
